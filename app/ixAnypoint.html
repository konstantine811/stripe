ixAnypoint

ixAnypoint infrastrucutre includes

1. Integration with all genetic services

  • Integrated with Helix sequencing services & APIs
  • Integrated with Helix mobile SDK
  • Integrated with physician-networks services & APIs
  • Integrated with genetic counselors services & APIs

2. Integrated with Apple ecosystem

  • Integrated with Apple IAP
  • Integrated with Apple Health-kit
  • Integrated with Apple Research-kit
  • Open Source Cocoapods libraries with extensive development components for genetic applications

3. End to end backend solution

  • Configurable to integrate with your backend technology
  • Configurable to or to be used be used as standalone end-to-end backend solution for your web and mobile application
  • Removing the need to run and maintain your own servers and databases.
  • REST API solution compatible with all exciting frontend stack (React, Angular, Vue, ...)

4. Multiple products solution

  • Business services to support multiple products with in the same platform or genetic application
  • User and profile linking across multiple genetic products or application
  • Single authentication across multiple genetic products or application

5. Genetic interpretation and reporting

  • Regulatory Compliance with CLIA for lab developed tests reporting
  • Regulatory Compliance with HIPAA for health information management
  • Hosting your interpretation and content for genetic reporting
  • API integration with your genetic interpretation and content environment
  • API integration with 3rd party genetic interpretation and content environments
  • Standalone hosting for genetic interpretation, content and reporting

6. Open Source plugins

  • Open source integration plugins to launch genetic product on WordPress, Drupal, or Joomla

7. Operation & Monitoring

  • All application on ixLayer infrastructure runs on a secure, virtual private cloud, isolated from any other application
  • All application on ixLayer infrastructure have isolated private services such as Caching, Queuing and Database Servers and Logging
  • All infrastructure are managed and monitored by ixLayer staff on a 24/7 basis.
  • The operation and monitoring includes, platform updates, integration and certificate management, source code updates, monitoring & logs, security patching and more

8. Security

  • ixLayer platform has been designed and developed with the strictest security and encryption practices.
  • Fully compliant with Industry security guidelines. TLS 1.2 security and AES-256 encryption. Encryption of Data in Transit and at Rest. Cloud security services.
  • Managed detection/response. Incident reporting. 3rd party security audits
  • ixLayer platform undergoes quarterly security, penetration and compliance testing

9. Update & improvements

  • ixLayer is an always improving infrastructure. We provided versioned non-breaking API changes to let you upgrade on your schedule without surprises
  • As an always improving infrastructure we work hard to make sure that we are always in sync with all of our services such as Helix genetic pipeline changes, regulatory updates, new IOS releases, etc...

10. Clinical Research

  • ixLayer infrastructure can also be used for running a clinical research study in your application.
  • Leverage the platforms integration with apple Health-kit and Research-kit
  • Launch new surveys or choose from validated tools
  • Visualize the phenotype and genotype data and discover possible correlations
